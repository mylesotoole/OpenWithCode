### Dependencies:
- [Visual Studio Code](https://code.visualstudio.com/) must be installed.

<hr>

### How to use:

3. Select ```Open With Code```.

### How to remove:
- Copy and paste the following command in your terminal:
```
rm -rf ~/Library/Services/Open\ With\ Code.workflow/
```
